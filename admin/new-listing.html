Inventory helper

Draft a shop item.json without memorizing the fields.

This page is static. It cannot upload images, authenticate to GitHub, or commit files. Use it to draft JSON, then upload or commit the item folder in GitHub.

Suggested folder

inventory/shop/new-listing/

Put this preview in item.json. Upload images into the same folder as front.jpg, back.jpg, detail-1.jpg, or add explicit image paths to the JSON.

After the files are committed, GitHub Actions rebuilds data/shop.json on push and on the 10-minute schedule.
